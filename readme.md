extremly incomplete wolfenstein port to the ti-83 plus using [z88dk](https://github.com/z88dk/z88dk)

## building
⚠️ building has only been tested using linux, you might experience problems building on other platforms⚠️

first, make sure your installation of [z88dk](https://github.com/z88dk/z88dk) and zcc is in your `PATH`.<br>
clone this repo:<br>
```
git clone https://github.com/CmosTheBattery/wolfensteinTI.git
cd wolfensteinTI
```
then just run `make` to build
